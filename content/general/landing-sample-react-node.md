## Week 1
### Focus
React fundamentals + modern JavaScript
### Key topics
- ES modules, closures, async/await, error handling
- Components, props, state, and the rendering mental model
- Core hooks: useState, useEffect, and side-effect patterns
### Deliverable
Mini dashboard app with list, filters, and loading/error states.
### Interview checkpoint
Explain render/re-render behavior and when useEffect should or should not run.

## Week 2
### Focus
Advanced React + global state
### Key topics
- Custom hooks, composition, and feature-based structure
- Forms, validation, and accessibility
- Global state and caching (Context or a lightweight alternative)
### Deliverable
Complete feature with form flows, validations, and local/remote persistence.
### Interview checkpoint
Discuss trade-offs between local state, Context, and server/cache-driven approaches.

## Week 3
### Focus
Node.js and REST APIs
### Key topics
- Layered architecture: routes, services, repositories
- Payload validation and centralized error handling
- Authentication with JWT/session and middleware
### Deliverable
User API + auth with secure endpoints and consistent error responses.
### Interview checkpoint
How do you design robust and versionable HTTP contracts?

## Week 4
### Focus
Data and persistence
### Key topics
- SQL modeling, indexes, relationships, and constraints
- Transactions, idempotency, and basic concurrency handling
- Migrations and seed data for dev/test environments
### Deliverable
Order module with transactions and an auditable event trail.
### Interview checkpoint
When should you use transactions, and how do you avoid inconsistent states?

## Week 5
### Focus
End-to-end React + Node integration
### Key topics
- Full frontend-backend flow with loading and optimistic updates
- Network error handling, retries, and empty-state UX
- Baseline security: CORS, rate limits, and sanitization
### Deliverable
Production-like flow (CRUD + auth + resilient UI error handling).
### Interview checkpoint
How do you design resilient UX when the backend partially fails?

## Week 6
### Focus
Testing and quality
### Key topics
- Unit tests in frontend and backend
- Integration tests for critical business routes
- E2E smoke tests for the main user journey
### Deliverable
Minimal test suite for login, create/update, and critical workflows.
### Interview checkpoint
Testing pyramid: what do you prioritize to deliver value quickly?

## Week 7
### Focus
Performance and observability
### Key topics
- React optimization: practical memoization and bundle splitting
- Node performance: profiling, caching, and I/O bottlenecks
- Structured logs, metrics, and baseline alerting
### Deliverable
Simple metrics dashboard and applied performance checklist.
### Interview checkpoint
How do you detect and fix a real production bottleneck?

## Week 8
### Focus
System design + interview readiness
### Key topics
- Design a content platform with queues and horizontal scaling
- Project storytelling: context, decisions, and trade-offs
- Technical + behavioral mock interviews with feedback
### Deliverable
System design doc plus STAR response set for behavioral rounds.
### Interview checkpoint
Present a fullstack architecture and justify each key decision.
